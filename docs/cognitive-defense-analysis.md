We do not evaluate the validity of specific conspiracy theories. We treat every documented technique as a potential attack vector and engineer defenses accordingly.

## Part 4: Uncovered Cognitive Dimensions

### 4.1 Proprioception and Vestibular Sense

**Role in identity**: The body schema — knowing where your limbs are without looking — is foundational to the pre-reflective sense of "I am here."

**Attack vectors**: Galvanic vestibular stimulation shifts perceived uprightness. Conflicting visual-vestibular input destabilizes embodied identity. Sensory deprivation dissolves body boundaries.

**Proposed defense**: Bone-conduction audio below ~800 Hz couples mechanically into the vestibular apparatus (the saccule responds to these frequencies). A spreading code modulated onto low-frequency tones could tag the vestibular channel. Accelerometer data from the phone (already collected) could feed back as haptic pulses synchronized to the code via the Vibration API, closing a proprioceptive loop.

**Implementation**: Add a sub-200 Hz oscillator to the audio engine. Modulate its frequency with a second Gold code channel. Add `navigator.vibrate()` pulses synchronized to chip transitions.

### 4.2 Olfactory / Gustatory (Proust Channel)

**Role in identity**: Olfaction bypasses the thalamus and projects directly to amygdala and hippocampus — the only sense with this privileged memory pathway. Odor-evoked memories carry stronger emotional charge than any other modality.

**Attack vectors**: Anosmia correlates with depersonalization. Synthetic odor environments can implant false contextual memories.

**Proposed defense**: Pair a specific physical scent (a chosen essential oil) with signal activation. The olfactory-hippocampal circuit conditions to associate that odor with the cryptographic identity. The scent becomes a non-digital second factor grounded in the deepest memory system.

**Implementation**: Add an onboarding prompt: "Choose a scent to pair with your identity. Apply this scent each time you use TINNITUS." Log scent-pairing sessions in the seed crystal. This is a protocol, not code — but the system should instruct the user and track compliance.

### 4.3 Interoception (Heartbeat, Breath, Visceral Sense)

**Role in identity**: Heartbeat awareness, breath rhythm, and visceral sensation constitute the "material me." Higher interoceptive accuracy correlates with more stable self-identity.

**Attack vectors**: False heartbeat biofeedback shifts emotional attribution. Breath-frequency entrainment can induce dissociation or panic.

**Proposed defense**: If wearable data is available (Web Bluetooth heart rate monitors), modulate the spreading code's chip rate to the user's cardiac frequency. The signal is only decodable by someone who knows the user's heartbeat rhythm — an attacker would need skin contact. The audio envelope can carry a subliminal amplitude modulation at cardiac frequency, reinforcing interoceptive-signal coupling.

**Implementation**: Add optional Web Bluetooth heart rate monitor pairing. Drive the coarsest fractal code level's period from cardiac R-R interval. Fall back to breath-rate detection via microphone amplitude envelope if no wearable is available.

### 4.4 Temporal Perception

**Role in identity**: Subjective time structures narrative identity. "Who I am" is inseparable from "when I feel I am." The subjective "now" window is ~2-3 seconds.

**Attack vectors**: Sleep deprivation collapses temporal discrimination. Flicker at specific frequencies alters time perception. Chronobiological disruption degrades executive function.

**Proposed defense**: The fractal code hierarchy already operates across timescales. Extend it to entrain circadian markers — a code epoch that aligns with the user's personal circadian phase (detectable via device usage patterns or skin temperature). An attacker without circadian-phase data cannot synchronize.

**Implementation**: Track session start times across days. Derive a circadian phase estimate from the pattern. Modulate the coarsest meta-layer hopping interval to align with the user's personal biological day. Add epoch markers to the worldpath that encode circadian phase.

### 4.5 Social / Mirror Neuron System

**Role in identity**: Self-other distinction depends on the mirror system and temporo-parietal junction. Identity is partly constituted by recognizing where "I" end and "you" begin.

**Attack vectors**: Synchronized movement and virtual avatar embodiment blur self-other boundaries. Deepfakes exploit face-processing trust.

**Proposed defense**: Gaze-contingent patterns — visual elements that transform based on where the user looks — would create a signal coherent only from the identity-holder's viewing angle. Requires front camera eye-tracking (available on modern phones).

**Implementation**: Use the front camera for basic gaze direction estimation. Shift the visual pattern's phase based on detected gaze angle. The pattern is only coherent from the user's actual viewing position.

### 4.6 Language / Inner Speech

**Role in identity**: The narrative self runs on inner speech and semantic memory. Disruption of inner speech fragments identity.

**Attack vectors**: Subvocal priming, semantic satiation, adversarial language models interfere with internal narration. Voice cloning attacks the auditory identity channel.

**Proposed defense**: The user's voice print can authenticate identity. A spoken passphrase, captured and hashed, becomes a biometric seed that binds the linguistic self to the cryptographic identity. The rotating phrase system already exists — make it speakable and voice-verified.

**Implementation**: Add optional voice enrollment via microphone. Hash the voice spectral fingerprint into the identity seed. On subsequent sessions, prompt for spoken phrase and verify against enrolled voiceprint.

### 4.7 Motor / Procedural Memory

**Role in identity**: How you walk, type, swipe — motor signatures persist through amnesia. Procedural memory is the most durable memory system.

**Attack vectors**: Transcranial stimulation can alter learned sequences. Fatigue erodes motor signatures.

**Proposed defense**: Keystroke dynamics and touch gesture patterns are already established behavioral biometrics. These can continuously feed the entropy accretion engine.

**Implementation**: Capture touch event sequences (position, pressure, timing deltas) and compute a gesture fingerprint. Accrete this into the identity — the worldpath becomes literally shaped by how the user moves.

### 4.8 Emotional Regulation

**Role in identity**: Baseline arousal, emotional reactivity, and recovery speed define temperament — the affective core of identity.

**Attack vectors**: Pharmacological manipulation, sustained stress, and algorithmic content targeting shift emotional baselines.

**Proposed defense**: Galvanic skin response and heart-rate variability index autonomic state. These can modulate signal parameters so the signal is only decodable by someone who knows the user's emotional signature.

**Implementation**: If wearable data is available, derive arousal estimate from HRV. Modulate the visual color temperature and audio modulation depth with arousal state. The signal's characteristics become coupled to the user's emotional baseline.

### 4.9 Sleep Architecture

**Role in identity**: Memory consolidation during slow-wave sleep and emotional integration during REM maintain coherent autobiographical self across days.

**Attack vectors**: Sleep deprivation is the most reliable identity destabilization method short of drugs. Targeted memory reactivation during sleep can alter memory content.

**Proposed defense**: The signal system could run a slow "sleep epoch" code during detected sleep periods, with parameters derived from sleep-stage transitions (detectable via accelerometer stillness and heart rate). This creates cryptographic continuity spanning the conscious/unconscious boundary.

**Implementation**: Detect likely sleep onset from accelerometer (sustained stillness) and time of day. Run a background audio tone at minimal volume with a unique sleep-epoch Gold code. On waking, the system verifies continuity between the sleep-epoch code and the waking code.

### 4.10 Gut-Brain Axis

**Role in identity**: The enteric nervous system and vagus nerve influence mood, decision-making, and personality traits.

**Attack vectors**: Antibiotics, diet shifts, and vagus nerve stimulation alter gut-brain signaling.

**Proposed defense**: Most distant from digital intervention. The only practical bridge: vagus nerve activity influences heart-rate variability, which is measurable. Gut-brain state enters the system indirectly through interoceptive channels.

**Implementation**: No direct mechanism. HRV-based interoceptive coupling (4.3) partially covers this surface indirectly.

## Part 6: What Cannot Be Defended Digitally

Some attack surfaces require non-digital countermeasures:

1. **Neurochemistry** — No digital signal can prevent pharmacological disruption. Defense requires physical security (preventing drug administration) or rapid detection (wearable toxicology — speculative).

2. **Physical sensory deprivation** — If the device is removed, the signal stops. Defense: the identity must persist in memory (the seed crystal concept), and the scent pairing provides a non-digital anchor that survives device confiscation.

3. **Extreme neurological disruption** (ECT, TBI) — If memory systems are physically destroyed, no external signal can restore them. Defense: the seed crystal is exportable and can be restored to a new device. The identity exists in the math, not just the brain.

4. **Social isolation** — If all social connections are severed, the identity signal can maintain self-reference but cannot substitute for human connection. Defense: the identity mesh (seed exchange between parties) creates mutual entropy commitments that are verifiable even after separation.
